**HOME-CREDIT-DEFAULT-RISK** 
An API that predict whether a loan applicant will default on their loan or pay back their loan. Home_credit_default_risk API is now live on render, you can view it through this link.

https://home-credit-default-risk.onrender.com/

#### Endpoint:
- GET /    → give you app info
- GET / health  → check if the model are loaded and check if the app is healthy
- GET/ features  → gives you all the columns names
- GET /docs  → gives you an interactive API documentation to interact with

#### HoW to Use it
click on the POST request, then click on try it out and paste the applicant data, then click on excute to get your answer.
